Importing the .txt File
1
Never double-click. Use File → Open.
Note (Windows): Toggle to All Files (*.*) instead of All Excel Files (bottom right) to find your .txt file. Select it → Text Import Wizard opens.
2
Step 1 → Delimited  |  Step 2 → check Tab only, uncheck all others.
3
Step 3 → click the Id column → set to Text.
4
Click on Advanced and set Decimal separator to . and Thousands separator to , and then click Finish.
5
Accept the Convert prompt when it appears.
Data Cleaning

Genes with too few reads have NA in all statistics columns. Because NA is text, it will break your number filters. Do this before you start analyzing:

1
Click the filter drop down on any of the statistics columns and uncheck NA.
2
Press Ctrl+A to select all the visible, clean data, then Ctrl+C to copy.
3
Open a new worksheet tab at the bottom (+) and press Ctrl+V.
4
Work exclusively from this new tab! Your raw data stays safe on Tab 1, and your number filters will work perfectly here.

Filtering for Significant Genes

Enable filters: click header row → Data → Filter or Ctrl+Shift+L

GoalHow to filter
Statistical significance padj dropdown → Number Filters → Less Thanpadj < 0.05standard  padj < 0.01stringent
Upregulated genes log2FoldChange → Number Filters → Greater Than 1> 1 = more than 2×
Downregulated genes log2FoldChange → Number Filters → Less Than -1< -1 = less than 0.5×
Both directions at once log2FoldChange → Number Filters → Between…
Mac: Select Between directly from the Filter menu.
Set: is greater than 1  OR  is less than -1
⚠ Make sure OR is selected, not AND
Recommended shortlist:  padj < 0.05  +  |log2FoldChange| > 1 — apply both filters together
